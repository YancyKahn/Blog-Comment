#CodeForces 805A

  一道水题, 很好想到
 > *    if l == r: <br>
 >          ans = l
 > *    else: <br>
 >          ans = 2


```
#include <cstdio>
#include <iostream>
using namespace std;
typedef long long ll;
int main() {
  ll a, b;
  while (cin >> a >> b) {
    printf("%lld\n", a == b ? a : 2);
  }
  return 0;
}
```
